# AI-Powered Banking Loan Risk Analytics System

## Project Overview

This project is a professional Banking Data Analytics and Machine Learning system developed using Python, SQL, Power BI, and Scikit-learn.

The main objective is to analyze banking customer data, identify risky customers, and predict loan default probability using Machine Learning algorithms.

# Key Insights

* Customers with low credit scores are more likely to default.
* High debt-to-income ratio increases risk.
* Self-employed customers show higher default rates.
* Large loan amounts increase risk probability.
